# SQLite
参考文档：

- [SQLite Home Page](https://www.sqlite.org/index.html)

SQLite 完全兼容 SQL 98 标准，使用标准 SQL 操纵数据库。
SQLite 是一个款轻量级的文件数据库，通常用于 Android 和 iOS 端应用内嵌数据库。


## 包管理器安装
```bash
sudo apt install -y sqlite3
```
```bash
sqlite3 *****.sqlite
```
